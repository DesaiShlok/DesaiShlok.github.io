Songbird Nocturnal Activity Detection

A sub-1GHz RF tracking system for behavior analysis of songbirds, developed as a Graduate Research Assistant at Columbia University's Internet Real-Time Lab in collaboration with Prof. Henning Schulzrinne. The project involves designing and deploying 70 custom RF sensor nodes for continuous, multi-month real-time telemetry in outdoor environments.

Key Highlights

Custom Altium PCB Design

Translated complex electrical and sensor requirements into a production-ready Altium PCB layout optimized for compact outdoor field deployment.

Ultra-Low Power Firmware

Bare-metal firmware utilizing deep-sleep states to achieve multi-month runtime on a single battery charge for continuous monitoring in the field.

70 Units Fabricated

Designed PCBs and fabricated 70 sub-1GHz RF nodes, each validated for reliable performance in outdoor environments.

Real-Time RF Telemetry

Sub-1GHz RF communication for low-power, long-range data transmission of bird activity metrics back to a base station.

Technologies
